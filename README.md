# XROO · 연오랑과 세오녀 위치기반 미션 (사이트 5개)

기획안에 따라 만든 **퀴즈 사이트 5개**입니다. 외부 라이브러리 없이 HTML 한 파일로
동작하므로(오프라인 가능), QR코드로 바로 연결해 쓸 수 있어요. 정답/오답 시 실제
XROO 씬으로 이동하도록 링크가 모두 연결되어 있습니다.

```
XROO/
├─ site1/index.html   ← 씬 1  · 마을 이름 (객관식)
├─ site2/index.html   ← 씬 3  · 해 日의 음 (주관식)  + 첫번째 힌트
├─ site3/index.html   ← 씬 6  · 대장간 (주관식)      + 두번째 힌트
│   └─ images/        ← 힌트 사진 3장 넣는 곳 (없으면 이모지 표시)
├─ site4/index.html   ← 씬 9  · 바위 이름 (객관식)
├─ site5/index.html   ← 씬 12 · 한 해 완성 글자 (객관식) + 세번째 힌트
└─ README.md
```

## 최종 미션은 "일월대" — 세 힌트가 모입니다

| 사이트 | 힌트 | 글자 |
|---|---|---|
| site2 | 첫번째 힌트 | **일 (日)** |
| site3 | 두번째 힌트 | **대** |
| site5 | 세번째 힌트 | **월 (月)** |

→ 日 + 月 + 대 = **일월대** 🎉

## 사이트 3 힌트 사진

`site3/images/` 폴더에 `hint1.jpg`(메달), `hint2.jpg`(아이스크림), `hint3.jpg`(다람쥐)를
넣으면 자동 표시됩니다. **없으면 이모지(🥈 🍓 🐿️)가 대신 보여서 그대로도 작동**합니다.

## 배포 방법 (택1)

- **그냥 열기:** `index.html`을 더블클릭하면 바로 열립니다.
- **무료 호스팅:** GitHub Pages / Netlify / Vercel 등에 각 폴더를 올리면 URL이 생깁니다.
  그 URL을 QR코드로 만들어 각 장소(씬)에 배치하세요.

## 디자인 메모 (기획안 반영)

- 글자 뒤 반투명 패널로 **가시성** 확보, "!!최종 미션 힌트!!"는 **빨간 글씨**
- `word-break: keep-all` + `clamp()`로 **모바일 잘림 방지** (375px 가로 오버플로 0 확인)
- 문장을 짧게·비슷한 길이로 끊어 배치
- 밤하늘·달 배경(별 자동 생성, 달 글로우)으로 설화 분위기 연출
